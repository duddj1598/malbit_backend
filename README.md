# Getting Started

This project is a starting point for the Spring Boot-based main server of the Malbit project. It handles business logic, user authentication, and data management.

A few resources to get you started if this is your first Spring Boot project:

* [Official Spring Boot Documentation](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/)
* [Spring Data JPA Reference Guide](https://docs.spring.io/spring-data/jpa/docs/current/reference/html/)
* [JUnit 5 User Guide](https://junit.org/junit5/docs/current/user-guide/)

For help getting started with Spring Boot development, view the online documentation, which offers tutorials, samples, guidance on backend development, and a full API reference.

## Project Specification
* **Main Language**: Java 21
* **Framework**: Spring Boot 3.2.4
* **Database**: MySQL 8.0
* **Build Tool**: Gradle
* **Architecture**: RESTful API Server (Planned integration with **Python-based AI Engine**)
